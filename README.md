# Projeto Teoria dos Grafos

## Tecnologias Utilizadas

Java  
JavaFX  

## Descrição

Este projeto aborda a implementação dos conceitos básicos de Teoria dos Grafos, com foco na modelagem, visualização e representação de grafos direcionados e não direcionados.

A aplicação permite trabalhar com diferentes formas de representação de grafos, especialmente por meio de matrizes, integrando uma interface gráfica desenvolvida em JavaFX para facilitar a visualização das estruturas.

## Funcionalidades

Implementação de conceitos fundamentais de grafos  
Criação e manipulação de grafos direcionados  
Criação e manipulação de grafos não direcionados  
Representação de grafos por meio de matrizes de adjacência  
Visualização gráfica utilizando JavaFX  

## Objetivo

O objetivo do projeto é consolidar o entendimento dos fundamentos de Teoria dos Grafos, explorando tanto a representação computacional quanto a visualização interativa dessas estruturas.
